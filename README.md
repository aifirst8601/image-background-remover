# image-background-remover

AI powered image background removal using BiRefNet.

High accuracy background removal for product photos, portraits, and e-commerce images.

## Features

- High precision background segmentation
- Transparent PNG output
- Support for various image formats
- Easy to use inference script

## Installation

```bash
git clone https://github.com/aifirst8601/image-background-remover.git
cd image-background-remover
pip install -r requirements.txt
```

## Usage

```bash
python inference.py --input input.jpg --output output.png
```
